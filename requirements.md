# ERD Requirements

This ERD visualizes the database entities for the Airbnb Clone:
- User: Manages profiles and roles.
- Property: Listings with details like price and location.
- Booking: Reservations linking users and properties.
- Review: Feedback on properties.
- Payment: Transactions for bookings.

Relationships: One-to-many (User to Property/Booking, Property to Review/Payment).
